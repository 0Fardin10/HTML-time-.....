## 🚀 How to Run

No setup needed. Just:

1. Clone the repository
   ```bash
   git clone https://github.com/0Fardin10/HTML-time-.....git
   ```
2. Open any `.html` file directly in your browser
   ```bash
   cd HTML-time-.....
   ```

Or simply open the `.html` files with **VS Code + Live Server** extension.
